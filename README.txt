Программа генерирует множество возможных решений (маршрутов), каждое из которых соответствует определенному времени и стоимости. Затем программа сравнивает эти решения между собой и выбирает те, которые не уступают другим решениям по обоим критериям одновременно. Эти решения называются "недоминированными" или "Парето-оптимальными".

После того, как программа определила все Парето-оптимальные решения, она отображает их на графике. Ось X графика соответствует времени, а ось Y - стоимости. Каждая точка на графике представляет собой одно из решений, а множество всех точек образует линию, называемую "фронтом Парето". Фронт Парето представляет собой множество всех недоминированных решений.

В этой программе всегда отображается только один лучший вариант, потому что она выбирает решение с наименьшим временем и наименьшей стоимостью из всех Парето-оптимальных решений. Это решение называется "идеальным решением" или "глобальным оптимумом".
